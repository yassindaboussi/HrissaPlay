🌟 Features

🎵

Live Radio Streaming

Stream multiple Tunisian radio stations with high-quality audio. Direct access to your favorite stations with one click.

🎨

Modern Design

Beautiful glassmorphism UI with smooth animations. Dark theme optimized for comfortable listening at any time.

📱

Fully Responsive

Works seamlessly on desktop, tablet, mobile, and even widget-sized windows. From 260px to 4K displays.

🎚️

Audio Visualizer

Real-time wave visualizer synced with audio playback. Dynamic colors and smooth animations enhance the listening experience.

🔄

Auto-Update

Automatic update checking with beautiful progress UI. One-click installation with auto-restart. Stay always up-to-date.

🌍

Multi-Platform

Native support for Windows, macOS, and Linux. Professional installers for each platform with seamless experience.

🔊

Volume Control

Precise volume slider with percentage display. Quick mute/unmute functionality with memory of previous volume level.

📡

Playback Controls

Play, pause, stop, and restart buttons for full control. Real-time status indicators for connection state monitoring.

🌙

Lightweight & Fast

Minimal resource usage for smooth performance. Quick startup time and instant streaming with no buffering.

🔐

Open Source

100% open source on GitHub. Community-driven development with regular updates and improvements.

🎯

Intuitive Interface

User-friendly design with no learning curve. Clear station selection and simple controls for all users.

📊

Status Indicators

Real-time connection status with color-coded indicators. Live, loading, error, and stopped states clearly shown.
